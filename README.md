2. Entrenamos la red neuronal (he aumentado las épocas) para mejor precisión.

3. Ya podemos utilizar la red neuronal entrenada 
  Nos arrojará el resultado, el cual será: Tipo de estrella y la probabilidad de confianza de la respuesta que nos da la red.
